# SaleCalc

## Setup and Run

```
//install packages
npm install
//launch project locally
npm start
//build the project
npm run build
```

## Install Playwright

```
//install browser binaries
npx playwright install
```

## Install EsLint

```
//init eslint
npx eslint --init
```
